# 📊 광고업 주요 상장사 재무 분석 대시보드

이 프로젝트는 파이썬을 활용해 대한민국 대표 광고 기업들의 **주가 데이터와 재무 건전성**을 실시간으로 분석하는 웹 서비스입니다.

## 🛠 사용 기술
- **Language**: Python 3.x
- **Framework**: Streamlit
- **Data**: yfinance (주가), Open DART API (재무)
- **Deployment**: Streamlit Cloud

## 🚀 주요 기능
1. **실시간 데이터 캐싱**: `@st.cache_data`를 활용해 반복적인 API 호출 없이 빠른 데이터 로딩 구현.
2. **수익성 지표 분석**: 매출 대비 영업이익률($$영업이익/매출 \times 100$$) 자동 계산 및 비교.
3. **인터랙티브 대시보드**: 사이드바 필터를 통해 사용자가 원하는 기업만 선별하여 분석 가능.
