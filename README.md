# 📊 대한민국 산업 분석 & 재무 건전성 대시보드

> **회계적 관점에서 산업별 핵심 지표(ROE, 부채비율)와 시장 트렌드를 통합 분석하는 플랫폼입니다.**

## 🚀 주요 기능
- **멀티 산업 분석**: 광고, 반도체, 게임(컴투스 등), 타이어 업종 데이터 탑재.
- **재무 건전성 시각화**: 영업이익률뿐만 아니라 **ROE(경영 효율성)**, **부채비율(안정성)** 지표 제공.
- **인터랙티브 차트**: Plotly를 활용하여 마우스 호버 시 기업별 주가를 직관적으로 비교 가능.
- **실시간 뉴스**: 네이버 뉴스 크롤링 및 IP 차단 대비 Fallback 버튼 시스템 구축.

## 🛠 Tech Stack
- **Frontend/Backend**: Streamlit
- **Data**: yfinance, BeautifulSoup4, Pandas
- **Visualization**: Plotly, Matplotlib

## 💡 회계팀 취업 포인트
- "단순 실적 나열이 아닌 부채비율과 ROE를 통해 기업의 재무 리스크와 자본 효율성을 분석하는 역량을 대시보드로 구현했습니다."
- "데이터 수집 과정에서의 기술적 결함(IP 차단 등)을 사용자 친화적인 버튼 시스템으로 해결한 경험이 있습니다.".
