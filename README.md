# 📊 산업별 재무 건전성 및 이익의 질(Quality of Earnings) 분석 대시보드

> **회계적 관점에서 기업의 장부상 이익과 실제 현금 흐름을 대조하여 '이익의 질'을 판별하는 전문 재무 분석 도구입니다.**

## 🚀 서비스 핵심 가치
본 프로젝트는 단순히 매출과 이익을 나열하는 수준을 넘어, 회계팀의 핵심 업무인 **리스크 관리**와 **현금흐름 분석** 역량을 시각화하는 데 중점을 두었습니다. 

## 🔗 Live Demo
[내 대시보드 접속하기](https://ad-dashboard-advtop5.streamlit.app/)

## ✨ 주요 분석 지표 (Accounting Focus)
1. **영업현금흐름(OCF) 분석**: 장부상 영업이익과 실제 유입된 현금을 대조하여 수익의 실현 가능성 검토.
2. **현금창출력(%)**: `(영업현금흐름 / 영업이익) * 100` 공식을 통해 이익의 질을 수치화. (100% 상회 시 우수한 이익의 질로 판단)
3. **재무 안정성 & 효율성**: 부채비율(안정성) 및 ROE(자본 효율성)를 통한 입체적 기업 평가.
4. **인터랙티브 주가 차트**: Plotly를 활용해 기업별 주가 추이를 직관적으로 비교 분석.

## 🛠 Tech Stack
- **Frontend/Backend**: Streamlit
- **Visualization**: Plotly (Interactive Chart), Matplotlib
- **Data Analysis**: Pandas, yfinance
- **Web Scraping**: BeautifulSoup4 (실시간 뉴스 연동)

## 💡 인사이트 및 문제 해결 (Troubleshooting)
- **데이터 구조 설계**: 업종별 감가상각비 특성(제조업 vs 서비스업)을 고려하여 영업이익과 현금흐름의 상관관계를 데이터베이스화함.
- **UX Fallback 설계**: 크롤링 차단 시 사용자가 즉시 대응할 수 있도록 '네이버 뉴스 직접 검색' 버튼 시스템을 구축하여 서비스 연속성 확보.
