# 📊 대한민국 주요 산업 분석 및 재무 건전성 대시보드

> **실시간 뉴스, 주가 트렌드, 핵심 재무 지표(ROE, 부채비율)를 통합 분석하는 인터랙티브 대시보드입니다.**

본 프로젝트는 단순한 데이터 나열을 넘어, 기업의 **수익성, 안정성, 효율성**을 다각도로 분석하여 회계/재무적 관점에서의 인사이트를 제공하기 위해 제작되었습니다.

## 🔗 서비스 바로가기
[내 대시보드 확인하기](https://ad-dashboard-advtop5.streamlit.app/)

## ✨ 핵심 기능
- **업종별 데이터 필터링**: 광고업, 반도체, 게임, 타이어 등 주요 산업군 선택 및 기업 간 비교분석.
- **재무 건전성 분석**: 회계팀 핵심 지표인 **부채비율(안정성)** 및 **ROE(효율성)** 시각화.
- **실시간 데이터 연동**: `yfinance` 기반 실시간 주가 차트 및 `BeautifulSoup` 기반 최신 뉴스 크롤링.
- **UX 최적화**: 서버 IP 차단 상황을 고려한 뉴스 직접 검색 버튼 시스템(Fallback 로직) 구현.

## 🛠 사용 기술
- **Language**: Python 3.x
- **Library**: Streamlit, Pandas, Matplotlib, yfinance, BeautifulSoup4, Requests
- **Deployment**: Streamlit Cloud

## 📈 회계팀 지원을 위한 재무 지표 선정 이유
1. **ROE (자기자본이익률)**: 투입된 자본 대비 이익 창출 능력을 파악하여 기업의 경영 효율성을 분석했습니다.
2. **부채비율**: 산업별 특성에 따른 재무 구조 차이를 파악하고 기업의 리스크 관리 능력을 모니터링합니다.
3. **영업이익률**: 본연의 사업 경쟁력을 확인하기 위해 매출액 대비 수익성을 분석 항목에 포함했습니다.

## 💡 주요 문제 해결 사례 (Troubleshooting)
- **이슈**: 클라우드 서버 환경에서 포털 사이트의 크롤링 차단(IP Block) 발생.
- **해결**: 
  1. HTTP Header(User-Agent) 최적화를 통한 우회 시도.
  2. 크롤링 실패 시 해당 기업 뉴스 검색 결과로 즉시 연결되는 **'인터랙티브 검색 버튼'**을 도입하여 사용자 경험(UX)의 연속성을 유지함.
