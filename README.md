# 📊 산업별 재무 건전성 및 이익의 질(Quality of Earnings) 자동화 대시보드

> **회계적 관점의 현금흐름 분석부터 '자동 PDF 리포팅'까지, 실무 프로세스를 그대로 구현한 전문 재무 분석 플랫폼입니다.**

## 🚀 프로젝트 목표 및 가치
본 프로젝트는 단순한 재무 지표 시각화를 넘어, 회계/재무팀의 핵심 업무인 **1) 현금흐름 기반의 리스크 탐지**와 **2) 경영진 보고를 위한 산출물(Deliverable) 자동화** 역량을 증명하기 위해 기획되었습니다.

## ✨ 핵심 기능 및 실무 적용 포인트

### 1. 현금흐름 기반 '이익의 질' 분석 (Accounting Focus)
- 장부상 영업이익과 실제 기업에 유입된 **영업현금흐름(OCF)**을 대조 분석합니다.
- 자체 지표인 **'현금창출력(OCF/영업이익)'**을 수치화하여 흑자 도산 리스크나 매출채권 회수 지연 등의 재무적 위험을 시각적으로 탐지합니다.

### 2. 자동 PDF 리포트 생성 (Reporting Automation)
- 대시보드 상에서 분석된 핵심 재무 데이터(ROE, 부채비율, 현금창출력 등)를 원클릭으로 추출합니다.
- `fpdf2` 라이브러리를 활용해 타 부서 및 경영진에게 즉시 공유 가능한 **A4 규격의 PDF 보고서**를 자동으로 렌더링하고 다운로드합니다.

### 3. 인터랙티브 시장 지표 트래킹
- `Plotly`를 활용하여 여러 기업의 1년 주가 추이를 마우스 호버(Hover)만으로 직관적으로 비교할 수 있는 인터랙티브 차트를 제공합니다.
- 네이버 뉴스 크롤링을 통해 기업별 최신 이슈를 실시간으로 모니터링합니다.

## 🛠 Tech Stack
- **Web Framework**: Streamlit
- **Data Analysis**: Pandas, yfinance
- **Visualization**: Plotly, Matplotlib
- **Document Automation**: fpdf2 (PDF 리포트 생성)
- **Web Scraping**: BeautifulSoup4, Requests

## 💡 주요 문제 해결 사례 (Troubleshooting)
1. **보고서 폰트 깨짐 현상 해결**: PDF 자동 생성 시 한글 폰트(`NanumGothic`) 인코딩 오류를 방지하기 위해 폰트 경로 매핑 및 Fallback 로직을 적용하여 안정적인 문서 출력을 구현했습니다.
2. **크롤링 IP 차단 대응**: 클라우드 서버 환경에서의 뉴스 크롤링 차단 시, 사용자에게 '직접 검색 버튼'을 렌더링하는 UX를 통해 서비스 연속성을 확보했습니다.

## 🎯 회계/재무 직무 역량 어필
